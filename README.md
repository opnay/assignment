# 깃허브 레포지토리 즐겨찾기

## 설정

[React Native Guide 문서](https://reactnative.dev/docs/environment-setup)를 기반으로 sdk 설치가 필요합니다.

ruby, android sdk (API Level 30) 설치

```shell
npm install
cd ios && pod install
```

## 실행방법

```shell
# Android
npm run android

# iOS
npm run ios
```

자동으로 개발 서버가 실행되나, 문제가 있을시 `npm run start`로 수동으로 실행.
